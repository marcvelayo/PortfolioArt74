Project 1- Glitch Art

Glitch Art was made using the Text Editor application and Audacity.

The galaxy was the original photo used for the Glitch Art. I liked this piece because it shows a lot of difference between the two photos. The photo edited using Audacity created a major texture change. It altered the whole photo, resulting in a gray-like pattern. The photo edited using Text Auditor showed a much clearer photo of the galaxy. It still contains the original colors, but the altered code created blurry and pixelated boxes.
